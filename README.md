# コンピューターグラフィックス特論 グループ課題

## 環境構築
### gitの使い方
必要に応じてSSH接続 \
https://zenn.dev/kazuki_tam/scraps/ff3f1f78cc9d81 (参考)

リポジトリのクローン \
`git clone git@github.com:ryotaro-iwata/ACG-Final-Report-2025.git`

### Nodejsのセットアップ
[参考](https://zenn.dev/karaage0703/articles/ecf24124c1a535) 

NVMのインストール 
```
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash
``` 
Nodejsのインストール \
`nvm install --lts`

### Nodejs の使い方
リポジトリの先頭ディレクトリへ移動 \
依存関係のインストール \
`npm install`

DEV環境の立ち上げ \
`npm run dev`
